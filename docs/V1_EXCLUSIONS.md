# Kobafin V1 Exclusions

This document lists features explicitly **not supported** in V1.

## Not Supported in V1
- Auto-rebalancing
- Asset switching inside pools
- Yield on BTC / ETH / SOL
- LPs or staking
- Leverage
- Shorting
- Price predictions

## Positioning
KobaFin is a self-custodied savings platform with optional crypto exposure. It is **not** an investment advisor, trading platform, or fund manager.

## Required Disclosures
- Crypto assets are volatile
- Only USDC earns yield in V1
- Yield is variable and not guaranteed
- Portfolio value may fluctuate
- No auto-rebalancing in V1
